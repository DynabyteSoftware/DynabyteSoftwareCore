◆ endLogger()

void Log::endLogger ( )
static

Stops logging and reverts clog to its default buffer

Note
Not to be used to pause logging. After endLogger() has been called, if a log file is assigned with the same filename as a previous log file, data will not be appended to it, the file will be overriden. This is done so as to not grow the log file unnecessarily. If the developer intends to keep log files indefinitely, they should include date and time in the filename.
