ContainerTransformWrapper< Container, ValueType > DynabyteSoftware::Collections::IEnumerable< Container, IteratorType >::select ( const typename TransformEnumerator< ValueType, iterator >::transform_function &  transform)

Returns the values of this container such that they are transformed in the manner according to the transform function provided

Parameters
transformThe transform function that determines how the values are to be transformed
Returns
A wrapper container that iterates through the contents of this container transforming any values within in the manner specified by the transform function
Note
The transformation evaluation does not occur immediately. Instead the container wrapper and iterators are created such that a lazy evaluation is performed when the wrapper is iterated
The wrapper returned by this function is valid for as long as the container is valid. It holds a reference to the container, not a copy, so changes to the container affect the results of the wrapper.
