Welcome

This is the webpage for our C++ core utilities library. Currently it provides a better Exception base class that holds some information about when it was thrown and a simple logging framework.

Logging

The logging framework replaces the stream buffer for std::clog upon initialization. So using it is as simple as including the "Logging.h" file and calling

Log::getLogger();

Optionally, you may assign a file where the log entries will be written to.

Log::getLogger().assignLogFile("file.log");

Once that's done, you can simply use clog as normal, and optionally provide it with a log level.

clog << Log::Information << "This is an information log entry" << endl;
clog << Log::Warning << "This is a warning log entry" << endl;
clog << Log::Error << "This is a error log entry" << endl;

Each log level could be assigned to a different log file, if desired.

Log::getLogger().assignLogFile(Log::Information, "information.log");
Log::getLogger().assignLogFile(Log::Error, "error.log");

Finally, you can clean up / close files at the end

Log::endLogger();

Exception

The provided Exception class holds some additional information about the location it was thrown, including the function name, source file name and line number.

These values are automatically populated using the provided THROW macro. Using it is as easy as including the "Exception.h" header and throwing an exception using the following macro.

THROW(Exception, "Exception message")

The exception will be self-logged when thrown. If a Log has been initialized and assigned a file to the Error log type, every exception will be automatically logged to that file.
